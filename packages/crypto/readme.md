# autism-crypto

autism-crypto is a crypto utility for autism. 

It supports the following crypto type
- ECDSA key gen, sign, verify
- ECDH key gen, derive
- AES encrypt, decrypt
- ZK rln, semaphore

## Install as NPM Package

```
npm install @autismjs/crypto
```

## Development

```
npm install
npm run lint
npm test
```

## Build

```
npm install
npm run build
```
